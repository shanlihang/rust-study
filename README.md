## Git 规范

#### 提交格式

| 类型     | 描述                                        |
| -------- | ------------------------------------------- |
| feat     | ✨ 新功能                                   |
| fix      | 🐛 修复问题                                 |
| docs     | 📖 文档更新                                 |
| style    | 🎨 代码格式调整                             |
| refactor | 🔨 代码重构                                 |
| perf     | ⚡ 性能优化                                 |
| test     | ✅ 测试相关                                 |
| build    | 📦 构建相关                                 |
| ci       | 🚀 持续集成相关                             |
| chore    | 🛠️ 其他杂项（构建、CI/CD 变更、依赖升级等） |
| revert   | 🔙 回滚代码                                 |
| merge    | 🔗 合并代码                                 |

#### 分支命名

> 分支名格式：类型/描述
